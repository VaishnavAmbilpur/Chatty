# Chatty

Chatty is a lightweight, anonymous real-time chat application where users can **create rooms**, **join existing rooms**, and **chat anonymously**.  
A room automatically **self-destructs when all participants leave**, ensuring privacy and temporary conversations with no stored history.

This project demonstrates real-time communication using WebSockets, dynamic room management, and a clean, minimal chat interface.

---

## 🚀 Features

- **Create a Room**  
  Generate a unique room ID and instantly start chatting.

- **Join a Room**  
  Enter an existing room using its ID and join the conversation anonymously.

- **Anonymous Chatting**  
  No login or personal information required.

- **Room Auto-Destruction**  
  When the last user leaves a room, the room is deleted from memory.

- **Real-Time Messaging**  
  Messages update instantly for all participants in the room.

- **Lightweight UI**  
  Clean and simple chat layout for fast interaction.
